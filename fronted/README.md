# 科研实习供需智能匹配与管理系统

一个现代化的科研实习项目管理平台，支持学生、教师和管理员三种角色，提供智能匹配、项目管理、申请审批等功能。

## 功能特性

### 🎓 学生端
- **项目浏览** - 查看所有可申请的科研实习项目
- **智能匹配** - 基于技能标签的 AI 智能项目推荐
- **在线申请** - 一键申请感兴趣的项目
- **申请管理** - 查看申请状态和进度
- **进度跟踪** - 实时查看项目进展

### 👨‍🏫 教师端
- **项目发布** - 创建和管理科研实习项目
- **申请审批** - 审核学生申请，通过或拒绝
- **进度管理** - 更新和跟踪项目进度
- **反馈评价** - 为学生提供评分和反馈

### 👑 管理员端
- **用户管理** - 管理所有用户和角色权限
- **数据统计** - 查看系统整体运行数据
- **项目监控** - 监控所有项目和申请状态
- **系统分析** - 关键指标和运营分析

## 技术栈

- **前端框架**: React 18 + TypeScript
- **样式方案**: Tailwind CSS v4
- **UI 组件**: Radix UI + shadcn/ui
- **状态管理**: React Hooks
- **图标库**: Lucide React
- **通知提示**: Sonner
- **构建工具**: Vite

## 后端 API

系统使用 RESTful API 与后端通信，基址为：`http://localhost:8080/api`

### 认证
- POST `/auth/register` - 用户注册
- POST `/auth/login` - 用户登录
- GET `/me` - 获取当前用户信息

### 用户管理
- GET `/users` - 获取用户列表
- GET `/users/:id` - 获取用户详情
- PUT `/me` - 更新个人信息

### 项目管理
- GET `/projects` - 获取项目列表
- POST `/projects` - 创建项目
- PATCH `/projects` - 更新项目
- DELETE `/projects` - 删除项目
- POST `/projects/archive` - 归档/恢复项目

### 智能匹配
- GET `/matches` - 获取智能匹配结果

### 申请管理
- POST `/apply` - 提交申请
- GET `/applications` - 获取申请列表
- GET `/applications/mine` - 获取我的申请
- POST `/application/status` - 更新申请状态

### 进度跟踪
- POST `/tracking` - 添加进度
- GET `/tracking` - 获取进度列表

### 反馈评价
- POST `/feedback` - 提交反馈

### 管理员功能
- GET `/admin/stats` - 系统统计
- POST `/admin/user/role` - 更新用户角色

## 使用说明

### 注册和登录

1. 访问应用首页
2. 点击"立即注册"创建账户
3. 填写姓名、邮箱、密码和角色
4. 添加技能标签（推荐）
5. 注册成功后自动登录

### 学生使用流程

1. **浏览项目** - 在"浏览项目"页面查看所有可申请的项目
2. **智能匹配** - 使用"智能匹配"功能获取 AI 推荐项目
3. **提交申请** - 点击"申请项目"按钮提交申请
4. **跟踪状态** - 在"我的申请"页面查看申请状态
5. **查看进度** - 通过后在"进度跟踪"查看项目进展

### 教师使用流程

1. **发布项目** - 在"我的项目"页面发布新项目
2. **编辑项目** - 随时编辑项目信息、要求和标签
3. **审批申请** - 在"申请审批"页面审核学生申请
4. **更新进度** - 为通过的申请添加进度记录
5. **提供反馈** - 为学生提供评分和评价

### 管理员使用流程

1. **用户管理** - 查看所有用户，修改用户角色
2. **数据监控** - 查看系统统计和关键指标
3. **项目管理** - 监控所有项目状态
4. **申请管理** - 查看所有申请记录

## 功能亮点

### 🤖 智能匹配算法
系统使用后端 AI 算法，基于学生的技能标签和项目要求进行智能匹配，提供匹配分数和原因说明。

### 📊 实时统计
管理员可以查看实时的系统运行数据，包括用户分布、项目数量、申请状态等关键指标。

### 🎨 现代化 UI
采用 OA 系统风格的界面设计，提供流畅的用户体验和响应式布局，适配桌面和移动设备。

### 🔐 角色权限
严格的权限控制，不同角色只能访问对应的功能模块，确保数据安全。

### 📱 响应式设计
完全响应式的界面设计，支持各种屏幕尺寸，提供一致的用户体验。

## 注意事项

1. **API 连接** - 确保后端服务运行在 `http://localhost:8080`
2. **认证令牌** - 登录后 JWT token 会自动存储在 localStorage
3. **技能标签** - 学生建议添加技能标签以获得更好的智能匹配结果
4. **浏览器兼容** - 推荐使用 Chrome、Firefox、Safari 等现代浏览器
